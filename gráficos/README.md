# Gráficos

## Características de un gráfico 
* Título del gráfico
* Eje x ( dominio )
* Título del eje x
* Marcas horizontales (indicadores en eje x)
* Eje y
* Título del eje y
* Marcas verticales (indicadores en eje y)
* Función ( linea recta que existe en un gráfico )
